Lost in Orbit

Lost in Orbit is a 2D platformer game made in java as a 4 week project in the course DD1349 (projinda23) at KTH. Collaboration with Nima Cheraghi.

Lost in Orbit is a 2D platformer game where players navigate a solar system to reach their ship after colliding with an asteroid. Developed in Java using the Swing framework, it features enemies, collectibles, and nine levels across three planets. A checkpoint system lets players restart at the first level of a planet after defeat.

Java Swing Game

Kvalkollen

Streamlining Swimmer Qualifications for Swedish Championships using python for webscraping and data manipulation.

Kvalkollen is a website designed to simplify the compilation of swimmers' results and qualifications for various Swedish championships. The times are sourced from Tempus Open and are presented in a manner that makes it easy to identify achieved qualification standards.

Python HTML Pandas BeautifulSoup Flask
